# Wayfound Agent Skills

Open-source agent skills by [Wayfound](https://wayfound.ai), published to [ClawHub](https://clawhub.ai) for the [OpenClaw](https://github.com/openclaw/openclaw) ecosystem.

## Skills

### wayfound

AI agent supervision for OpenClaw. Makes your user's expectations explicit, persistent, and measurable through approved guidelines, automated session analysis, and continuous improvement.

- **Onboarding wizard** — Discovers existing sessions, proposes supervisors and guidelines, shows value before asking for configuration
- **Supervisors & guidelines** — Named sets of user-approved standards with configurable alert levels
- **Automated analysis** — Cron-based session evaluation with grading (Hot to go / Needs review / Needs attention)
- **Alerting** — Guideline violation alerts on any messaging channel, immediate or batched
- **Continuous improvement** — Suggested behaviors, knowledge gap tracking, and trend analysis over time

To install, just ask OpenClaw:

> "Install the wayfound skill from ClawHub"

Skill files: [`clawhub/`](clawhub/)

## About Wayfound

Wayfound is the world's first AI agent supervision platform. This repo contains free, open-source skills that bring core Wayfound concepts to individual users. For teams and organizations that need multi-agent fleet supervision, enterprise compliance, shared organizational learning, and more, visit [wayfound.ai](https://wayfound.ai).
